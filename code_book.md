CodeBook
========

###Columns Description

+	SUB - number of the subject input number
+	ACT - physical activity. Contains text values from this list:
  *	WALKING
  *	WALKING_UPSTAIRS
  *	WALKING_DOWNSTAIRS
  *	SITTING
  *	STANDING
  *	LAYING
+	OTHER COLUMNS - Remaing columns are measurements in the following format: 
  * _Type of measurement___Suffix_
  * i.e. fBodyGyro_std()-X
